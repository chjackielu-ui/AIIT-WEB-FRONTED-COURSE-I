HTML 列表

无序列表 (ul)

无序列表使用 <ul> 标签，每个列表项使用 <li> 标签。默认使用圆点作为标记。

  • 项目一
  • 项目二
  • 项目三
  • 项目四

有序列表 (ol)

有序列表使用 <ol> 标签，每个列表项使用 <li> 标签。默认使用数字作为标记。

  1. 第一步
  2. 第二步
  3. 第三步
  4. 第四步

有序列表的 type 属性

可以通过 type 属性设置有序列表的标记类型：

  1. 大写字母标记
  2. 大写字母标记
  1. 小写字母标记
  2. 小写字母标记
  1. 大写罗马数字标记
  2. 大写罗马数字标记
  1. 小写罗马数字标记
  2. 小写罗马数字标记

定义列表 (dl)

定义列表使用 <dl> 标签，每个定义项使用 <dt>（定义术语）和 <dd>（定义描述）标签。
